单列集合框架结构

  • Collection接口：单列集合，用来存储一个一个对象
    • List接口：存储有序的。可重复的数据 –>”动态”数组
    • ArrayList、LinkedList、vector
  • Set接口：存储无序的、不可重复的数据 –>高中讲的”集合”
    • HashSet、LinkedHashSet、TreeSet
