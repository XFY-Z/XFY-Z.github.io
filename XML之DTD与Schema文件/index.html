XML语义约束

  • XML文档结构正确，但可能不是有效的
  • 例如：员工档案XML中绝对不允许出现”植物品种”标签。XML语义约束就是用于规定XML文档中允许出现哪些元素
  • XML语义约束有两种定义方式：DTD与XML Schema

Doucument Type Defintion

  • DTD(Document Type Definition ,文档类型定义)是一种简单易用的语义约束方式

  • DTD文件扩展名为.dtd

XML Schema的概述
